# ÷ El Denominador

**Observatorio del denominador — liquidez global en tiempo real**

> Cada precio es una fracción. El numerador es el activo. El denominador es la cantidad de unidades monetarias en circulación. Cuando el denominador crece, el número sube — pero el valor real no cambia.

## ¿Qué trackea?

- **M2 Global** — Oferta monetaria de EE.UU., Eurozona, Japón y China
- **Balance de Bancos Centrales** — Fed, BCE, BoJ, PBoC
- **Liquidez Neta de la Fed** — Fed Balance Sheet − TGA − RRP
- **Índice Denominador** — Compuesto ponderado que mide la expansión del denominador

## Stack

- [Next.js 15](https://nextjs.org/) — App Router
- [Recharts](https://recharts.org/) — Visualización de datos
- [Tailwind CSS 4](https://tailwindcss.com/) — Estilos
- TypeScript
